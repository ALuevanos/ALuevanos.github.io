TrailCart Capstone Project

As the Project Manager, I led a team to develop a GPS-based food cart tracking system to support students.

Project Overview

This project addresses the challenge of locating food carts on campus, especially during peak hours. The system integrates GPS tracking with a touchscreen interface to provide real-time location and wait-time updates, improving accessibility and reducing stress for students.

System Components

  • Raspberry Pi for data processing and system control.
  • GPS module for real-time cart location tracking.
  • Touchscreen interface for managing wait times.
  • Backend server and API integration for mobile app display.

Challenges and Future Enhancements

Some challenges included weak cellular signals in certain areas and balancing performance with energy efficiency. Future plans involve implementing a dual-network system (Wi-Fi and cellular) for improved reliability and refining the mobile app interface based on user feedback.

Impact

This project contributes to:

  • Improved food accessibility for students.
  • Healthier eating habits through reduced stress.
  • Advancement of smart campus infrastructure.
