TrailCart Capstone Project

My Role: Project Manager & Lead Developer

Project Overview

I spearheaded a capstone project aimed at helping students locate campus food carts more easily. By integrating a GPS module, Raspberry Pi, and a touchscreen interface, I enabled real-time location and wait-time updates. This initiative reduces the stress of wandering around campus looking for food options—especially during peak hours.

Key Responsibilities & Actions

  • Coordinated Team Activities: I assigned tasks, set deadlines, and led weekly stand-ups to keep everyone aligned on deliverables.
  • Developed Hardware & Software Integration: I programmed the Raspberry Pi to collect data from a GPS module and display wait-time information on a touchscreen.
  • Implemented Backend Server: I built an API that captures cart locations and wait times, then updates a mobile app in real time.
  • Conducted User Testing: I organized prototype demos for students, gathered feedback, and iterated on the interface for improved usability.

Technical Skills Demonstrated

  • Python programming for Raspberry Pi and server-side logic
  • GPS module configuration and data parsing
  • Touchscreen UI design and implementation
  • RESTful API development and database integration
  • Project management, team leadership, and iterative prototyping

Current Status & Future Plans

  • Optimizing Connectivity: I’m exploring dual-network support (Wi-Fi and cellular) to maintain stable tracking in areas with weak signal.
  • Enhancing UI: I’m collecting more user feedback to refine the touchscreen interface and app layout for better accessibility.
  • Scaling for Multiple Carts: I plan to expand the system to track multiple carts simultaneously, ensuring the database and UI can handle increased traffic.

Key Outcomes

  • Improved Convenience: Students no longer waste time roaming campus; they get precise cart locations and wait-time alerts.
  • Reduced Crowds & Stress: Real-time information helps balance foot traffic, lowering wait times and congestion.
  • Foundational Smart Campus Tool: This project paves the way for other GPS-powered campus solutions.
