Random Number Generator

Interactive Dice Simulation Using Digital Logic and Soldering

Project Overview

This project involved designing and assembling a Random Number Generator that simulates dice rolls. The device was built using digital logic principles and implemented on a custom PCB (Printed Circuit Board).

Key Skills and Tools

  • Soldering components onto the PCB with precision.
  • Utilizing logic gates to implement random number generation.
  • Understanding digital circuit design and troubleshooting.
  • Testing and debugging using multimeters and other electronic tools.

Process

  1. Designed the circuit using digital logic and determined component placements on the PCB.
  2. Assembled the PCB by soldering logic gates, resistors, LEDs, and buttons.
  3. Validated the design by testing the random number generation functionality with live circuits.
  4. Troubleshot and optimized the circuit to ensure accurate and consistent performance.

Outcome and Impact

  • Successfully implemented a functioning Random Number Generator with an interactive LED display.
  • Enhanced my hands-on experience with soldering and circuit design.
  • Demonstrated expertise in digital logic principles and troubleshooting techniques.
