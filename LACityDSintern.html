Data Science intern - Earn Learn Play Project

Interactive map development for the City of Los Angeles

Project Overview

As part of the UCR DS-PATH Fellowship, I worked with a team on the Earn Learn Play project in partnership with the City of Los Angeles. The goal of this project was to create an interactive map that helps families find free or low-cost programs, services, and events for youth. This map is live and available on the Earn Learn Play website.

    Key Responsibilities

    • Developed Interactive Maps: Created and updated the map using ArcGIS Online to visualize youth programs across the city.
    • Streamlined Data Pipelines: Programmed Python scripts to clean and standardize data, ensuring accurate updates from Google Sheets to ArcGIS.
    • Database Management: Designed a MySQL database to organize and store large datasets for seamless integration.
    • Enhanced User Experience: Improved search and filtering features to allow users to find programs based on criteria like location, age, and cost.

    Challenges and Solutions

    • Data Quality: Addressed inconsistencies in department-provided datasets by implementing validation checks and automated cleaning processes.
    • Automation: Reduced manual updates by automating data uploads to ArcGIS, improving efficiency.

    Technical Skills Demonstrated

    • Python for data cleaning, scripting, and automation
    • Google Sheets for collaborative data management
    • MySQL for database creation and integration
    • ArcGIS Online for mapping and visualization

    Impact

    • Made programs and resources more accessible to families across Los Angeles
    • Improved data reliability through automation and validation
    • Enhanced user experience with refined search functionality
