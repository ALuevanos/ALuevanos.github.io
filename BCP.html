Breast Cancer Stage Prediction

Machine learning experimental project using Random Forest and Support Vector Machine

Project Summary

This project aims to predict breast cancer stages (I-IV) using machine learning models like Random Forest and Support Vector Machine. The goal is to enhance early detection and personalized treatment, reducing mortality rates.

Dataset and Methodology

The dataset sourced from SEER contains 4,024 records with 16 attributes, including tumor size, hormone receptor status, and grade. Models were trained and tested with an 80-20 split, using stratified cross-validation to ensure balanced class distributions.

Key Results

  • Random Forest achieved 100% accuracy, making it ideal for clinical applications.
  • SVM achieved 99.75% accuracy, handling complex patterns effectively.
  • Both models demonstrated high precision and recall across all classes.

Impact

The project highlights the potential of machine learning in healthcare, providing accurate staging predictions that can inform treatment plans and improve patient outcomes.
